# ReLens Slidev Style

## Theme

Use the default theme:

```md
---
theme: default
---
```

## Color Palette

Only 3 colors:

| Color | Hex | Usage |
|-------|-----|-------|
| Background | `#fff4dc` | Warm cream background |
| Primary | `#fe8630` | Orange accent, emphasized words |
| Secondary | `#000000` | Black text |

## Typography

- Body text: default system font (black)
- Emphasized words in titles: MonteCarlo (Google Font, orange)

## Usage Guidance

- Keep a light background with dark text (no dark theme)
- Use the orange accent sparingly for highlights
- Keep contrast high for legibility

## Title Emphasis Pattern

Use MonteCarlo for important words in the title:

```md
# Become <span style="font-family: 'MonteCarlo', cursive; color: #fe8630;">Agent Native</span> company

# Make <span style="font-family: 'MonteCarlo', cursive; color: #fe8630;">America Great</span> again
```
